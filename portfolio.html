Portfolio Projects

The projects seen below showcase the experience I have gained various web development technologies.

Simon demonstrates my knowledge of web development were I utilized HTML/CSS/Javascript to created a web-browser game based on the electronic board game Simon.

Hyrule-Compendium demonstrates my knowledge on building a modular and modern front-end application utilizing React.js and organizing and displaying a vast amount of data from a 3rd party API to the front-end application. I chose to work on a front-end application that would display the game compendium for users of a popular Nintendo Switch Game Zelda Breathe of the Wild.

Flex-Snooze demonstrates my ability to work as a productive team member within a small team of developers. The task was to build a MERN Stack application in a small team environemnt. My primary tasks in the project revolved around providing the functionality of the Add Custom Workout Component and CSS styling to ensure that the app had mobile first functionality and to ensure the app had a good user experience.

Quest-Log demonstrates my ability to build a Full-Stack application from the conception to a complete product. I utilized a MERN stack to construct the application. The the app's main function consists of having users create pins on a world map where the user can save their personal thoughts and experiences about different locations around the world.
